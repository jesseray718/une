# Credit Alliance

**Unionized, formula-driven credit & resource pool for underprivileged nodes.**

- 50-person modular hubs that operate as one unit
- PoPW-verified contribution builds credit/reputation
- Tier system based on consistency + mutual support
- Formula (`cooperation_efficiency` + `unified_effective_power`) used to allocate roles that maximize value per node while caring for the whole
- Permaculture principles: capture energy, stack functions, self-regulate, integrate rather than segregate

## v0.1 Goals
- Contribution logging
- Credit calculation using UNE formula
- Basic hub matching
- Tier system

Future: Smart contract simulation, mobile interface, integration with H-003/AeroCement builds.
